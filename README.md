# 🎬 Video-Streaming-Project

## 📌 Project Overview

The **Video Streaming Management System** is a database management project developed using **MySQL** to manage and organize operations of a digital video streaming platform.

The system automates important OTT platform activities such as:

- User Management
- Subscription Handling
- Video Content Organization
- Payment Tracking
- Watch History Maintenance

This project focuses on designing a structured relational database with proper relationships between tables using **Primary Keys** and **Foreign Keys**.

It also demonstrates practical implementation of SQL operations including:

- CRUD Operations
- JOIN Queries
- Subqueries
- Aggregate Functions

---

## 🎯 Project Objectives

- Design and implement a relational database for a video streaming platform
- Manage users, subscriptions, videos, genres, and payments efficiently
- Perform CRUD (Create, Read, Update, Delete) operations
- Execute advanced SQL queries using:
  - JOINs
  - Subqueries
  - Aggregate Functions
- Improve data consistency and accessibility through proper database design
